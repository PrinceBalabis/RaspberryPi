This is a simple voice control package.
A fork from http://blog.oscarliang.net/raspberry-pi-voice-recognition-works-like-siri/

Install
sudo chmod +x RaspberryPi/basicvoicecontrol/main.sh
sudo chmod +x RaspberryPi/basicvoicecontrol/speech2text.sh
sudo chmod +x RaspberryPi/basicvoicecontrol/text2speech.sh

Usage
RaspberryPi/main.sh

Example questions
Hello, What is your name?
How much is 3 squared?
Why did the chicken cross the road?
How many roads must a man walk down before you can call him a man?
